# HTML

Repository ini berisi kumpulan contoh file **HTML** untuk belajar dan menjelajah struktur halaman web, termasuk penggunaan tag dan semantic HTML.

---

##  Tentang
HTML (HyperText Markup Language) adalah bahasa markup utama untuk membuat struktur halaman web.  
Di repo ini kamu bisa melihat contoh berbagai struktur HTML seperti heading, paragraf, semantic tag, dan lainnya.

---

##  Struktur Folder
dasar/  
├── heading.html  
├── paragraf.html  
├── semantic.html  

---

##  Cara Menjalankan
1. Klik file `.html`
2. Pilih **View Raw** atau
3. Klik **Open in Browser** untuk melihatnya tampil di browser  

(Note: ini adalah file HTML statis, jadi cukup dibuka di browser biasa)
